## 🎨 Ancient Indian Aesthetic Plan

### 1. Color Palette - Traditional Indian

```css
/* Current: Modern grey/blue */
--primary: 240 5.9% 10%; /* Grey */

/* New: Ancient Indian */
--saffron: 30 100% 50%; /* Bhagwa - Spiritual */
--deep-red: 0 70% 40%; /* Spiritual power */
--gold: 45 100% 50%; /* Knowledge/wisdom */
--earth: 30 30% 30%; /* Ancient manuscripts */
--ink: 220 20% 15%; /* Traditional ink */
--parchment: 40 30% 90%; /* Old paper */
```

### 2. Typography - Sanskrit & Traditional

- Use **Google Fonts**: Noto Sans Devanagari, Yatra One, or Tillana
- Display Sanskrit prominently
- Traditional calligraphy for headers

### 3. Visual Elements

- **Lotus motifs** (spiritual purity)
- **Mandala patterns** (sacred geometry)
- **Parchment textures** (ancient manuscripts)
- **Om/Aum symbols** (subtle branding)
- **Diya/lamp icons** (enlightenment)

### 4. Terminology Changes

| Current (Generic) | New (Ancient Indian)       |
| ----------------- | -------------------------- |
| Library           | **Granthalaya** (ग्रंथालय) |
| AI Guide          | **Guru AI** (गुरु)         |
| Quiz              | **Pariksha** (परीक्षा)     |
| Verse             | **Shloka** (श्लोक)         |
| Chapter           | **Adhyaya** (अध्याय)       |
| Search            | **Anveshana** (अन्वेषण)    |
| Contents          | **Suchi** (सूची)           |
| Structure         | **Vinyas** (विन्यास)       |

### 5. Unique Features (Not in Tatva)

#### A. Audio Chanting

- Play Sanskrit shlokas with proper pronunciation
- Different voices (male/female/child)
- Adjustable speed

#### B. Meditation Timer

- Built-in meditation guide
- Bell sounds at intervals
- Breathing exercise animations

#### C. Daily Wisdom

- "Today's Shloka" feature
- Push notifications (if PWA)
- Calendar integration (festival dates)

#### D. Visual Stories

- Illustrated epics (Mahabharata, Ramayana)
- Interactive maps (ancient India)
- Family trees (lineages)

#### E. Mantra Chanting

- Popular mantras with audio
- Repetition counter
- Mala (rosary) tracker

### 7. Layout Changes

#### Navigation

- **Top**: Saffron/orange band with Sanskrit quote
- **Logo**: Stylized Om or Lotus
- **Menu**: Vertical sidebar (like ancient scrolls)

#### Cards

- **Parchment background** instead of white/grey
- **Gold borders** (traditional frame)
- **Sanskrit headers** with English subtitles
- **Illustrations** (lotus, swan, peacock motifs)

### 8. Content Differentiation

#### Tatva Focus:

- Reading and exploration
- Sanskrit translations

#### Hind AI Focus (Unique):

- **AI-Powered Learning** (our unique feature)
- **Interactive Quiz** (gamification)
- **Voice Search** (speak in Sanskrit/Hindi)
- **Personalized Path** (AI-curated journey)
- **Community Features** (discuss, ask questions)
