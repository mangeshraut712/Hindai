# SanskritNova Merge Notes

SanskritNova has been consolidated into Hind AI as the Sanskrit Studio surface at `/sanskrit-nova`.

## Current Status

- Hind AI remains the canonical repository and runtime.
- Sanskrit learning, transliteration, guided tracks, and merged Sanskrit analysis tools live inside this app.
- Gemma 4 via OpenRouter remains the single AI runtime.
- Navigation, sitemap, and footer should point users to Hind AI routes only.

## Repository Cleanup

Before deleting the old SanskritNova repository, verify:

- The latest Hind AI branch is pushed to GitHub.
- SanskritNova-specific routes and learning features are present in Hind AI.
- Any issues, documentation, deployment settings, or secrets needed from SanskritNova are copied or no longer needed.
- A final archive or GitHub backup exists if you want historical access.

Once those checks are complete, delete the SanskritNova repository from GitHub settings.
