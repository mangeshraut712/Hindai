# 🚀 Hind AI Setup Guide for Hackathon Judges

Quick start guide to run Hind AI locally with **Gemma 4 via Ollama**.

## ⚡ 5-Minute Quick Start

### Step 1: Install Ollama

```bash
# macOS
brew install ollama

# Linux
curl -fsSL https://ollama.com/install.sh | sh

# Windows
# Download from: https://ollama.com/download/windows
```

### Step 2: Pull Gemma 4

```bash
ollama pull gemma4:4b
```

Expected output:

```
downloading gemma4:4b... 100% ▕██████████████████▏ 2.7 GB
```

### Step 3: Start Ollama

```bash
ollama serve
```

Server will start on `http://localhost:11434`

### Step 4: Clone & Run Hind AI

```bash
# Clone repository
git clone https://github.com/mangeshraut712/Hindai.git
cd Hindai

# Install dependencies
npm install --legacy-peer-deps

# Copy environment file
cp .env.example .env.local

# Edit .env.local to point to Ollama
# OLLAMA_URL=http://localhost:11434

# Start development server
npm run dev
```

### Step 5: Test Gemma 4 Integration

Open browser to `http://localhost:3000/ai-guide` and ask:

> "Explain Bhagavad Gita 2.47"

You should see a response generated by **Gemma 4 locally**.

## 🔧 Troubleshooting

### "Ollama connection refused"

Make sure Ollama is running:

```bash
ollama serve
# In a separate terminal window
```

### "Model not found"

Pull the model again:

```bash
ollama pull gemma4:4b
```

### "npm install fails"

Use legacy peer deps:

```bash
npm install --legacy-peer-deps
```

### "Port 3000 already in use"

Use different port:

```bash
npm run dev -- --port 3001
```

## 🧪 Testing Gemma 4 Directly

Test the model without Hind AI:

```bash
# Basic generation
curl http://localhost:11434/api/generate \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gemma4:4b",
    "prompt": "Explain the concept of Dharma in 3 sentences",
    "stream": false
  }'

# Chat mode
curl http://localhost:11434/api/chat \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gemma4:4b",
    "messages": [
      {"role": "system", "content": "You are a wise Guru."},
      {"role": "user", "content": "What is Karma Yoga?"}
    ],
    "stream": false
  }'
```
